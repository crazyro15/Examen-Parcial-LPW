¿Cuáles son los lenguajes de programación soportados por un navegador web?

Microsoft Edge, por ejemplo, funciona con un motor totalmente nuevo y requiere del uso de múltiples lenguajes de programación de forma regular. Entre ellos se incluyen C, C++, HTML5, JavaScript y Node.js.

Describa la arquitectura de 3 capas. Describir cada capa utilizando una lista HTML.

  • Capa de presentación (parte en el cliente y parte en el servidor).
    1. Recoge la información del usuario y la envía al servidor (cliente).
    2. Manda información a la capa de proceso para su procesado.
    3. Recibe los resultados de la capa de proceso.
    4. Generan la presentación.
    5. Visualizan la presentación al usuario (cliente).

  • Capa de proceso (servidor web).
    1. Recibe la entrada de datos de la capa de presentación.
    2. Interactúa con la capa de datos para realizar operaciones.
    3. Manda los resultados procesados a la capa de presentación.

  • Capa de datos (servidor de datos).
    1. Almacena los datos.
    2. Recupera datos.
    3. Mantiene los datos.
    4. segura la integridad de los datos.
